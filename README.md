# FlClashR 🇷🇺

**Свободный интернет в одну кнопку — VPN-клиент для России**

Форк FlClash, адаптированный для пользователей в России.
Простой интерфейс, настройки по умолчанию для обхода блокировок.

## Возможности

- 🚀 Один экран — четыре кнопки
- 🇷🇺 Пресет «Россия» — YouTube, Telegram, Gemini через VPN, банки напрямую
- 🔒 DNS через Cloudflare (1.1.1.1)
- 📋 Импорт подписки по ссылке
- 🌙 Тёмная тема, Material You дизайн

## Установка

Скачайте APK из раздела [Actions → Artifacts](../../actions)

## Сборка
flutter pub get
dart run build_runner build
flutter build apk --release
