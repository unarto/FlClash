# FlClashR 🇷🇺

**Свободный интернет в одну кнопку — VPN-клиент для России**

Форк FlClash, адаптированный для пользователей в России.

## Возможности
- 🚀 Один экран — четыре кнопки
- 🇷🇺 Пресет «Россия» — YouTube, Telegram через VPN, банки напрямую
- 🔒 DNS через Cloudflare (1.1.1.1)
- 📋 Импорт подписки по ссылке
- 🌙 Material You дизайн

## Установка
Скачайте APK из раздела [Actions → Artifacts](../../actions)
